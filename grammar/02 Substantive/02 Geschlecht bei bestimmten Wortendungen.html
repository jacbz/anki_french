Das Geschlecht bei bestimmten Wortendungen

Anhand der Wortendung lässt sich oft das Geschlecht eines Substantivs bestimmen. Die folgenden Listen zeigen typische männliche und weibliche Endungen.

Männliche Endungen

-aire le commissaire, le dictionnaire
der Kommissar, das Wörterbuch
-an (+ Kons) l'an, le sang
das Jahr, das Blut
-ent le vent, l'argent
der Wind, das Geld
Ausnahme: la dent
der Zahn
-ment le développement
die Entwicklung
-ier le métier, le cahier
der Beruf, das Heft
-et le billet, le guichet
das Ticket, der Schalter
-in le vin, le voisin
der Wein, der Nachbar
Ausnahmen: la fin, la main
das Ende, die Hand
-ain le pain, le bain
das Brot, das Bad
-ail le travail, le détail
die Arbeit, das Detail
-al l'hôpital, le cheval
das Krankenhaus, das Pferd
-isme le tourisme, le socialisme
der Tourismus, der Sozialismus
-eau le tableau, le bureau
das Bild, das Büro
Ausnahme: l'eau
das Wasser
-oir le devoir, le pouvoir
die Pflicht, die Macht
-teur le moteur, l'ordinateur
der Motor, der Computer
-age le garage, le visage
die Garage, das Gesicht
Ausnahmen: l'image, la cage, la page, la plage
das Bild, der Käfig, die Seite, der Strand
-ège le collège, le manège
die Mittelschule, die Manege

Im Deutschen sind Substantive auf -age und -ege weiblich (die Garage, die Manege), im Französischen sind sie jedoch männlich.

Weibliche Endungen

-ade la promenade, la salade
der Spaziergang, der Salat
-ance la distance, la confiance
die Entfernung, das Vertrauen
Ausnahme: le stade
das Stadion
-ence la différence, la concurrence
der Unterschied, die Konkurrenz
-ée la journée, l'employée
der Tag, die Angestellte
Ausnahmen: le musée, le lycée
das Museum, das Gymnasium
-elle la nouvelle, la chandelle
die Nachricht, die Kerze
-ère la bouchère, la panthère
die Metzgerin, der Panther
-esse la jeunesse, la politesse
die Jugend, die Höflichkeit
-ette la baguette, la cigarette
das Baguette, die Zigarette
-euse la vendeuse, la friteuse
die Verkäuferin, die Fritteuse
-ie la boulangerie, la maladie
die Bäckerei, die Krankheit
Ausnahmen: l'incendie, le comité
der Brand, das Komitee
-té la nationalité, la popularité
die Nationalität, die Beliebtheit
Ausnahmen: le pâté, l'été
die Pastete, der Sommer
-ine la cuisine, la semaine
die Küche, die Woche
Ausnahme: le domaine
der Bereich
-ion la conversation, la télévision
das Gespräch, der Fernseher
Ausnahmen: l'avion, le camion, le million
das Flugzeug, der LKW, die Million
-ure la nature, la voiture
die Natur, das Auto
Ausnahme: le murmure
das Murmeln
-tude l'habitude, l'inquiétude
die Gewohnheit, die Beunruhigung

Manche Substantive gibt es sowohl in männlicher als auch in weiblicher Form. Bei diesen Wörtern verbirgt sich hinter dem unterschiedlichen Geschlecht ein Bedeutungsunterschied, z.B.:

le tour / la tour
die Tour / der Turm
le mode / la mode
die Art und Weise / die Mode
