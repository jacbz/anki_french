Das Geschlecht der Substantive

Im Französischen gibt es nur männliche und weibliche Substantive. Das deutsche Neutrum existiert nicht. Das Geschlecht kann sich vom Deutschen unterscheiden (z.B. la mort – der Tod, le vase – die Vase).

Da man das Geschlecht oft nur am Artikel erkennt, sollte man Substantive immer mit ihrem Artikel und Geschlecht lernen.

Das Geschlecht bei Lebewesen

Bei Personen oder Tieren gibt es in der Regel für jedes Geschlecht eine eigene Form.

2. Gleiche Form für beide Geschlechter

Bei einigen Substantiven ist die Form identisch. Nur der Artikel zeigt das Geschlecht an:

un élève / une élève
ein Schüler / eine Schülerin
un enfant / une enfant
ein Kind (Junge/Mädchen)
un journaliste / une journaliste
ein Journalist / eine Journalistin
un secrétaire / une secrétaire
ein Sekretär / eine Sekretärin

Einige ehemals ausschließlich männlich verwendete Berufsbezeichnungen existieren nur in der Form des Maskulinums. Das Geschlecht wird durch den Artikel kenntlich gemacht:

un ingénieur / une ingénieur
ein Ingenieur / eine Ingenieurin
le médecin / la médecin
der Arzt / die Ärztin

3. Unterschiedliche Wörter für männlich und weiblich

Bei manchen Personen-, Verwandtschafts- oder Tierbezeichnungen gibt es komplett unterschiedliche Wörter:

un homme / une femme
ein Mann / eine Frau
un garçon / une fille
ein Junge / ein Mädchen
un frère / une sœur
ein Bruder / eine Schwester
un coq / une poule
ein Hahn / eine Henne

Das Geschlecht bei Sachen und Dingen

Bei Sachen und Dingen gibt es oft keine logischen Regeln, aber einige Wortgruppen haben tendenziell ein bestimmtes Geschlecht.

Männlich sind in der Regel:

  • Wochentage:
    le lundi (der Montag), le vendredi (der Freitag)
  • Jahreszeiten:
    le printemps (der Frühling), l'automne (der Herbst)
  • Himmelsrichtungen:
    le sud (der Süden), le nord (der Norden)
  • Sprachen:
    le portugais (Portugiesisch), l'italien (Italienisch)
  • Bäume:
    le chêne (die Eiche), le sapin (die Tanne)
  • Metalle & chemische Elemente:
    l'or (das Gold), le mercure (das Quecksilber)
  • Transportmittel:
    le bus, le train, l'avion

Weiblich sind in der Regel:

  • Länder (die auf ‑e enden):
    la France, la Pologne
    (Ausnahmen: le Portugal, le Danemark, le Luxembourg)
  • Flüsse:
    la Saône, la Moselle
    (Ausnahmen: le Rhône, le Danube)
  • Wissenschaften:
    la géographie, la médecine
    (Ausnahme: le droit - die Rechtswissenschaft)
  • Autonamen:
    la BMW, la Citroën
